# RC Car

---

Welp, this is a rc car that I built using two arduinos, one for the receiver and one for the transmitter.

What I've used:
   - 2 Arduino Nano 
   - 1 servo
   - 1 motor
   - L298n motor driver
   - an old rc car body
   - 2 joysticks (transmitter)
   - more little electronics
